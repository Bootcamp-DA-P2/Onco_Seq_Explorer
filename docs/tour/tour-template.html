01. Introducción

¿Qué es OncoLens y qué hace?

OncoLens es una aplicación web de machine learning

Diferencia muestras de tejido obtenidas en una biopsia, entre tejido sano y tejido tumoral, y posteriormente diferencia entre 5 tipos de cáncer: mama, riñón, pulmón, próstata, colon.

¿Cómo funciona?

El programa está preparado para que se le introduzca un documento xlsx, csv o tsv con la expresión génica RNA-Seq que predice si tiene cáncer o no y qué tipo de entre los 5 con los modelos entrenados.
